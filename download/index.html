Installation Instructions

Windows

  1. 1. Download the .exe installer
  2. 2. Run the installer (admin not required)
  3. 3. Follow the setup wizard
  4. 4. Launch Calibration Wizard from the Start menu or desktop shortcut

macOS

  1. 1. Download the .pkg installer
  2. 2. Open the package and step through the installer (approve if macOS prompts for security)
  3. 3. Launch Calibration Wizard from Applications or Launchpad

System Requirements

Minimum

  • •Windows: Windows 10 (64-bit) or later, ASIO compatible audio device
  • •macOS: macOS 11.0 Big Sur or later (Intel or Apple Silicon), Core Audio compatible audio device
  • •4 GB RAM
  • •500 MB free disk space
  • •Internet connection (for cloud calibration processing)

Recommended

  • •8 GB RAM or more
  • •1 GB free disk space
  • •Audio interface with measurement microphone
  • •High-speed internet for faster job processing
